Survey Form Application
Project Description:
This project is a Survey Form Application designed to demonstrate user interaction and data collection through a simple and responsive interface. Users can fill out survey questions, and their responses are securely stored in a database for further analysis.The application focuses on usability, form validation, and backend integration to handle.

Tools & Technologies Used
HTML – Structure of the survey form
CSS – Styling and responsive design
PHP – Server-side processing and form handling
MySQL – Database for storing survey responses
